== Description ==

Adder uses ISO-8859-2 encoding for tags, instead of UTF-8. This plugin fixes this.

[Changelog](http://wordpress.org/extend/plugins/adder-tags-fix/changelog/)

== Installation ==

1. Upload `adder-tags-fix` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enjoy :)
